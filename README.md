# AI Powered Image Resizer
A lightweight Python project that uses an LLM (Gemini) to analyse images to resize

## Features
- Pick images in a directory
- Analyze the image by using LLM (Gemini) according to the provided prompt.
- Resize the image to 50%

## Requirements

- Python 3.7 or later


## Installation

1. **Clone the repository:**

    ```
    https://github.com/ranwala/AI-Powered-Image-resizer.git
    cd AI-Powered-Image-resizer
    pip install -r requirements.txt
    ```

3. **How to run**
    ```
    python main.py
    ```

## Configuration
Create a .env file, and add below:
- GOOGLE_API_KEY=your_model_key
